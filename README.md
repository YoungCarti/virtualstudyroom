# Virtual Study Room App (Under Development)

A modern, feature-rich Flutter application designed as a student dashboard. This app helps students track assignments, view study progress, and access community resources with a premium, glassmorphism-inspired UI.

## üöÄ Features

*   **Modern UI/UX**:
    *   Sleek dark mode design with glassmorphism effects.
    *   Smooth animations and transitions.
    *   Custom gradient icons and buttons.
*   **Home Dashboard**:
    *   **Dynamic Greeting**: Personalized welcome message that updates with your name.
    *   **Rotating Quotes**: Motivational quotes that refresh every minute to keep you inspired.
    *   **Assignments Tracker**: View pending assignments with due dates and status indicators.
    *   **Feature Grid**: Quick access to "My Learnings", "My Community", "AI Tutor Guidance", and "Study Progress".
    *   **Floating Navigation**: A stylish pill-shaped bottom navigation bar.
*   **Profile & Settings**:
    *   **Dedicated Menu**: A separate, glass-styled menu for Profile, Account, and Settings.
    *   **Detailed Profile**: View full user details including name, email, phone, and bio.
    *   **Secure Authentication**: Integrated with Firebase Auth for secure login and logout.

## üõ†Ô∏è Tech Stack

*   **Framework**: [Flutter](https://flutter.dev/) (Dart)
*   **Backend**: [Firebase](https://firebase.google.com/)
    *   **Authentication**: Secure user sign-in/sign-up.
    *   **Cloud Firestore**: Real-time database for user profiles and app data.
*   **State Management**: `StatefulWidget` & `StreamBuilder` for real-time updates.

## üèÅ Getting Started

### Prerequisites

*   [Flutter SDK](https://docs.flutter.dev/get-started/install) installed.
*   A Firebase project set up with Authentication and Firestore enabled.

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/YoungCarti/virtualstudyroom
    cd project
    ```

2.  **Install dependencies**:
    ```bash
    flutter pub get
    ```

3.  **Firebase Setup**:
    *   Configure your Firebase project.
    *   Add your `google-services.json` (Android) and `GoogleService-Info.plist` (iOS) to the respective directories.

4.  **Run the app**:
    ```bash
    flutter run
    ```
